# MoneyManagerQt
Создать простой менеджер доходов/расходов, который позволит пользователям отслеживать ежедневные денежные доходы и расходы.
Он будет показывать общий доход, классифицировать расходы и баланс каждый месяц.
Он также имеет возможность хранить данные за каждый месяц и отображать годовые данные.
